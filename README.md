## Описание

Простой конвертер валют с графическим интерфейсом, использующий **фиксированные курсы** (не требует API-ключа).

## Особенности

- ✅ Работает без интернета
- ✅ Не требует API-ключа
- ✅ 25 популярных валют
- ✅ Сохранение истории в JSON
- ✅ Красивый интерфейс на Tkinter
- ✅ Актуальные курсы на Апрель 2026
